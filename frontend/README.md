# Cab Allocation System
This Web Application has build for both frontend and backend development

## Installation

sudo apt-get install curl
sudo apt-get install -y nodejs   
node --version  (13.6.0) 
npm --version (6.13.4)
sudo npm install -g create-react-app
npm start
